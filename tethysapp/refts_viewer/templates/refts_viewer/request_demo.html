    Demo & Help

    You can use this app to visualize a WaterML time series data hosted on HydroShare.

    1) Go to a HydroShare service instance (e.g. dev.hydroshare.org), find/create a HIS Time series resource.

    2) Fill the blanks on this page with the correct Resource ID and WaterML filename from HydroShare resource page.

    3) Click on "Show Chart (post request)".

    Next release will register this viewer into HydroShare system as a HydroShare App. Then you can launch this viewer from HydroShare resource page by a simple click.

    For more info, check our source code at GitHub
